# tannerholman.space

Personal site for Tanner Holman — coaching, personal training, and movement classes for meditators.

Static HTML/CSS/JS, no build step. Deployed via GitHub Pages from `main`, served at [tannerholman.space](https://tannerholman.space) (custom domain set via `CNAME`).

## Structure

```
index.html                  # home
coaching.html               # 1:1 coaching
pt-for-meditators.html      # personal training
dzogchen-movement.html      # sky realm series
body-mapping.html           # monthly class
styles/shared.css           # shared tokens + base styles
js/say-hi.js                # copy-email-to-clipboard behavior
assets/                     # images
```

## Local development

Open any `.html` file in a browser. No server needed.

## Deployment

Push to `main`. GitHub Pages rebuilds in ~30s.
